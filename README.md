# AutoCAD-AI

AI-assisted platform for converting photos and PDF drawings into editable 2D AutoCAD geometry.

## Initial Goal

Convert architectural plans, sketches, scans, and reference images into structured 2D CAD drawings.

## Planned Capabilities

* Upload PDF and image files
* Detect drawing scale
* Extract lines, polylines, arcs, circles, and text
* Recognize walls, doors, windows, rooms, and dimensions
* Preview and correct detected geometry
* Export editable DXF or DWG files
* Send geometry into AutoCAD through a controlled plugin
* Add 3D reconstruction in a later phase

## Current Scope

The first version will focus only on 2D drawings.
